  • HTML Tags are building blocks of HTML Page. HTML Elements are components that are used in HTML Page. HTML Tags usually exist in pairs consisting of a starting and an ending tag.

  • HTML tags are used to hold the HTML element. HTML element holds the content. HTML attributes are used to describe the characteristic of an HTML element in detail.

  • A void element is an element whose content model never allows it to have contents under any circumstances. Void elements can have attributes. For example br, img, hr ect

  • HTML entities are the reserved characters that are used in the HTML document

  • A Class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element within the page

  • Formatting elements were designed to display special types of text

  • Cell padding is used to create a border around the content area of a web page, whereas cell spacing is used for positioning elements within that content area.

  • We can club two or more rows or columns into a single row or column in an HTML table with help of "colspan" and "rowspan"

  • Inline elements never begin on a new line. Block elements completely fill the accessible area from left to right. Only the area defined by the tags in the HTML element is covered by inline elements. There are top and bottom margins for block elements.
